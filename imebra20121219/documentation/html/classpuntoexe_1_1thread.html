Detailed Description

This class represents a thread.

The thread can be started with thread::start(). Once the thread is started, the virtual function threadFunction() is executed in a separate thread.

The code in threadFunction() can call shouldTerminate() to know when the thread should be terminated gracefully as soon as possible.

The function terminate() can be called to inform the thread that it should terminate. terminate() is called automatically by the thread destructor which also waits for the thread termination before deallocating it and returning.
