std::wstring puntoexe::imebra::handlers::dataHandlerTime::getUnicodeString ( const imbxUint32  index) const [virtual]

Return a string representing the time stored in the buffer.

The returned string has the format: "HH:MM:SS.FFFFFF" where:

  • HH is the hour
  • MM is the minute
  • SS is the second
  • FFFFFF is the fraction of seconds in nanoseconds
