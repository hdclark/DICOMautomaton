Detailed Description

Represents a single waveform of a dicom dataset.

Waveforms are embedded into the dicom structures (represented by the dataSet class), stored in sequence items (one waveform per item).

Use dataSet::getWaveform() to retrieve a waveform from a dataSet.

imbxInt32  paddingValue = 0x7fffffff

Retrieve the decompressed waveform data.

Retrieve the requested channel's data and decompress it into signed long values. 8 bits uLaw and aLaw data are decompressed into normalized 16 bits values.

This function takes into account the value in the interpretation tag and returns an handler to ready to use data.

Parameters:
channelthe channel for which the data is required
paddingValuethe value that the function must write in the returned data in place of the original padding value. Specify a number outside the range -32768..65535
Returns:
a data handler attached to ready to use data. Use handlers::dataHandler->GetSignedLong() or handlers::dataHandler->GetSignedLongIncPointer() to retrieve the data
